# $groupAcceptInvite

Bot will be accept the given Group invite link and join the group.

### Usage:

```plain
$groupAcceptInvite[invite link]
```

<Callout>
The invite link must be URL like `https://chat.whatsapp.com/invitecode` not only invite code!
</Callout>
