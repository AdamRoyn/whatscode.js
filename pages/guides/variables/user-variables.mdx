# User Variables

Value will be stored in each user per group.

<Callout emoji="⚠️">
  Make sure you already have a variable added in `bot.variables`like in [First of All page](first-of-all)
</Callout>

### Set:

```plain
$setUserVar[name;value;group jid (optional if you in groups);user jid (optional)]
```

### Get:

```plain
$getUserVar[name;group jid (optional if you in groups);user jid (optional)]
```
