# Group Variables

Value will be stored for each group.

<Callout emoji="⚠️">
  Make sure you already have a variable added in `bot.variables`like in [First of All page](first-of-all)
</Callout>

### Set:

```plain
$setGroupVar[name;value;group jid (optional)]
```

### Get:

```plain
$getGroupVar[name;group jid (optional)]
```
