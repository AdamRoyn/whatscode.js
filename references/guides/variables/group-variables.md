# Group Variables

{% hint style="warning" %}
Make sure you already have a variable added in `bot.variables`like [.](./ "mention")
{% endhint %}

### Set:

```
$setGroupVar[name;value;group jid (optional)]
```

### Get:

```
$getGroupVar[name;group jid (optional)]
```
